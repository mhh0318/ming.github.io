Algorithms

K-Means Cluster is a kind of prototype-based cluster. Its basic idea is to find a division scheme for k clusters by iteration, so that the total error obtained when the mean value of these k clusters is used to represent the corresponding various samples is the minimum.

For a given dataset $D = \lbrace x_1,x_2,…,x_m \rbrace$, and the cost function of k-Means Cluster $C = \lbrace C_1,C_2,…,C_k \rbrace$ is based on minimum sum of squares errors. $$ E = \sum^k_{i=1} \sum_{x \in C_i} \left||x-\mu_i|\right|_2^2$$ is based on minimum sum of squares errors. where $\mu_i = \frac{1}{C_i} \sum_{x \in C_i} x$, $\mu$ is the average vector of $C_i$. The smaller the error is, the higher the sample similarity of a cluster will be.

Minimization the cost is a NP-hard problem, the common algorithm to find approximate solution is an iterative refinement technique with Greedy Strategy.

Pseudocode:

Input: Dataset $D = \lbrace x_1,x_2,…,x_m \rbrace$, Cluster Number $k$
Program:
Initialize $\mu$ by select $k$ data randomly from the dataset
Repeat
     set $C_i = \oslash$
     for $j=1,2,…,m$ do
         Calculate the distance of sample $x_i$ and each average vector$\mu_i$;
         According to the smallest distance set the label of sample;
         Append the sample into relevant cluster.
     end for
     for $i=1,2,…,k$ do
         Calculate updated average vector $\mu$;
         if $\mu’_i \ne \mu_i$ then
             Update current $\mu_i$ as $\mu’_i$;
         else
             Keep the average vector;
         end if
     end for
until No change in average vectors.
Output Clusters $C = \lbrace C_1,C_2,…,C_k \rbrace$
