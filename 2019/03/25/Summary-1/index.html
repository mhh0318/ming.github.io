This part I wanna write down some understand about 2-layers neural network, mainly about Logit Regression to do binary classification tasks and some derive of forward propagation and backward propagation.

Construct of Network

Cost : Logit Regression Model

In Logit Regression, we can get a prediction output $\hat{y}$. Assume our task is a supervised learning binary classification task, then we will have a true value $y$ and the value is $0$ or $1$, So our prediction value $\hat{y}$ is expected to fall in $[0,1]$, cause it represents the probability of $y$ when $x$ is given. We use a normal activation function - sigmoid function $\sigma$ to make the regression value in $[0,1]$.

Sigmoid Function

$$ \sigma(z) = \frac{1}{1+ e^{-z}} $$This function make a linear function change into the unlinear function and limit the output in $[0,1]$

Then we have the logit regression model and we can use this model to define a cost function for training the weight $\omega$ and bias $b$ $$ \hat{y} = \sigma(\omega^Tx+b) $$

Loss Function

Given $ { (x^{(1)},y^{(1)}),…,(x^{(m)},y^{(m)}) } $, we want to get $\hat{y} \approx y$, so we want to have a function to show when the loss is minimum.

Maybe MSE function is suitable?

Nope, MSE might give us too many “minimum” value due to it can be non-convex function. Therefore, we introduce another method cross-entropy or maximum likelyhood estimation to get the loss : $$ H(x) = - y\log\hat{y} $$ and I have to say we do binary classification now thus we have $0,1$ two conditions. So let me introduce the cross-entropy loss function $$ \ell = - (y\log\hat{y} + (1-y)\log(1-\hat{y})) $$

Here cross entropy have the same performance as Maximum Likelyhood Estimation, because the essence of Cross-Entropy is the maximun of likelyhood function in classification task

I will prove this cost function is corrcet in another mathematical way at the bottom of this post.

Cost Function

Loss Function is defined with respect to a single training example. It measures the performance on a single training example. And the cost function is defined to measure how well your entire training set. $$ J(w,b) = - \frac{1}{m} \sum_{i=1}^m \ell(y,\hat{y}) $$ $$ = - \frac{1}{m} \sum_{i=1}^m y^{(i)}\log\hat{y}^{(i)} + (1-y)^{(i)}\log(1-\hat{y})^{(i)} $$

Gradient Descent

Once we have cost function in the network, we want to minmize the cost, namely, we want to make $ J(w,b) $ as small as possible. Gradient Descent is a method to do that. Here we have a equation expressed the Gradient Descent: $$ \theta = \theta - \alpha \frac{dJ(w,b)}{d\theta} $$ Here $\theta$ can be $\omega$ or $b$. And $\alpha$ is the magic number or Learning rate. Normally it is a very small number due to our output may be always a number in $[0,1]$

Propagation in Network

Now we have a neural network as following:

Forward Propagation

In a shallow neural network, we suppose there are only one hidden layers with four units. If we have a input $X$ with three parameters $x^{(1)}, x^{(2)}, x^{(3)}$, and the size of it is $[3,1]$. In hidden layer, for example, the first unit of the layer, the input $$Z^{(hidden)(1)} = \omega^{(1-1)}x^{(1)} + \omega^{(2-1)}x^{(2)} + \omega^{(3-1)}x^{(3)} + b $$ We can change it into a vectorize version $$ Z^{(hidden)(1)} = W^{(1)T}X + b^{[1]} $$ and the output is the first unit input, so the size of $Z^{(hidden)(1)}$ is $[1,1]$

If we stack all $\omega$ together, then, in this example, the size of $W^{[1]}$ become $[3,4]$, so the output of $$ Z^{(hidden)} = W^{[1]T}X+b^{[1]} $$ become the input of all four units in the hidden layer, which size is $[4,1]$

Then we employ an activation function $g$ on the input $Z$, the activate function can be Sigmoid, tanh or ReLU and others. Here we use sigmoid function, the output of hidden layers $$ a^{(hidden layer)} = g(Z^{(hidden)}) = \sigma(Z^{(hidden)}) $$ Here we should know the size of $a^{(hidden layer)}$ is $[4,1]$.

Next is the output layer, as usual, $$Z^{(output)} = W^{[2]T}X + b^{[2]}$$ where the size of $W^{[2]T}$ is $[4,1]$ and the prediction, which is also the output of the output layer, $$ \hat{y} = a^{(output layer)} = g(Z^{(output)}) = \sigma(Z^{(putput)}) $$

Back Propagation

From Forward Propagation, we get the prediction $\hat{y}$ with some parameters $W^{[1]},b^{[1]},W^{[2]},b^{[2]}$. We also have the cost function $$ J(W^{[1]},b^{[1]},W^{[2]},b^{[2]}) = = - \frac{1}{m} \sum_{i=1}^m y^{(i)}\log\hat{y}^{(i)} + (1-y)^{(i)}\log(1-\hat{y})^{(i)} $$

From output layer to hidden layer, we can compute $$dZ^{[2]} = \frac{dJ}{dZ^{[2]}} = A^{(outputlayer)}-Y$$ so $$dW^{[2]} = \frac{dJ}{dW^{[2]}} = \frac{1}{m} dZ^{[2]} A^{(hiddenlayer)T}$$ and $$db^{[2]} = \frac{dJ}{db^{[2]}} = \frac{1}{m} dZ^{[2]} $$

From hidden layer to input layerm we can compute $$dZ^{[1]} = \frac{dJ}{dZ^{[1]}} = W^{[2]T}dZ^{[2]} * g^{[1]\prime}(Z^{(hidden)})$$ so $$dW^{[1]} = \frac{dJ}{dW^{[1]}} = \frac{1}{m} dZ^{[1]} X^{T}$$ and $$db^{[1]} = \frac{dJ}{db^{[1]}} = \frac{1}{m} dZ^{[1]} $$

Gradient Descent

Then we can set learning rate in Gradient Descent function:$$ W^{[1]} = W^{[1]} - \alpha \frac{dJ}{dW^{[1]}} $$ $$ b^{[1]} = b^{[1]} - \alpha \frac{dJ}{db^{[1]}} $$ and $$ W^{[2]} = W^{[2]} - \alpha \frac{dJ}{dW^{[2]}} $$ $$ b^{[2]} = b^{[2]} - \alpha \frac{dJ}{db^{[2]}} $$

Results

After a number of iteration, we can get a set of $W^{[1]},b^{[1]},W^{[2]},b^{[2]}$ which make the best prediction has the best fitting value.
